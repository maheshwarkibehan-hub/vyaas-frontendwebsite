ğŸ”„ Request Flow Example: "Aaj ka weather bata"

  1. User speaks: "Aaj ka weather bata"
  2. LiveKit captures: Audio stream browser se backend tak
  3. Gemini processes: Speech-to-text + Intent understanding
  4. Tool execution: vyaas_get_weather() function calls OpenWeatherMap API
  5. Response generation: Gemini creates natural Hinglish response
  6. Voice synthesis: Text-to-speech with "Charon" voice
  7. Delivery: Audio stream user ke browser tak
  8. Logging: Conversation save in database + activity log
